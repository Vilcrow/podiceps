Preferences

Menu → Edit → Preferences

Changeable settings:

Language

Interface → Language

Select desired language and click the Save button. Restart the program to apply the changes.

Theme

Interface → Theme

Select desired theme and click the Save button. You can set light, dark and custom theme. For a custom theme, specify the path to the qss file.

Table View

Table → Table View

You can show/hide the following table columns:

  • Transcription
  • Status
  • Date
