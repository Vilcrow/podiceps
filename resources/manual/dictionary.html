Dictionary

The dictionary stores all the words you entered. It cannot contain words with duplicate originals.

The dictionary is saved in xml format.

GUI

GUI of the dictionary consists of the following parts:

  • Table Widget
  • Input Widget
  • Find Widget
  • Add Widget
  • Edit Widget
  • Statistics Widget

Table Widget

Main Window

The Table Widget displays the contents of the dictionary.

The table consists of the following columns:

Original
A word in the source language. Max length: 50.
Transcription
Transcription of the word. Max length: 50.
Translation
Translation of the word. Max length: 100.
Status
The status of word learning.
Date
Date the word was added.

Input Widget

Main Window

With the Input Widget, you can quickly add a new word. Enough to specify the original of the word and click on the Add button.

You can also specify a transcription, translation and status for the word.

Find Widget

Menu → Edit → Find...

Input Widget → Find

With the Find Widget, you can set a filter for the Table Widget.

Enter a filter in the input line and click on the Set button. To reset the filter, clear the input and click on the Set button.

Add Widget

Menu → Edit → Add...

Table Widget → Right Click → Add...

The Add Widget is a window where you can specify a original, transcription, translation, status and comment for the word and add it to the dictionary.

Edit Widget

Menu → Edit → Edit...

Table Widget → Right Click → Edit...

Using the Edit Widget you can edit a group of words.

Select the necessary rows in the table and call the Edit Widget. If you don't select the rows then all rows will be selected.

In the window that appears, you can edit or delete a word. If necessary, you can recover the source value.

To save the change, click the Save button. To make all changes to the table, click the Record button. To undo all changes click the Cancel button.

Statistics Widget

Menu → Tools → Statistics

The Statistics Widget shows statistics on your dictionary.
