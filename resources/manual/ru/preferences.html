Настройки

Меню → Изменить → Настройки

Изменяемые настройки:

Язык

Интерфейс → Язык

Выберите нужный язык и нажмите кнопку Сохранить. Перезапустите программу для применения изменений.

Тема

Интерфейс → Тема

Выберите нужную тему и нажмите кнопку Сохранить. Вы можете установить светлую, тёмную или пользовательскую тему. Для пользовательской темы укажите путь до qss файла.

Вид таблицы

Таблица → Вид таблицы

Вы можете вывести/скрыть следующие столбцы:

  • Транскрипция
  • Статус
  • Дата
