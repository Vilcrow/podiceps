Словарь

Словарь хранит все введенные вами слова. Он не может содержать слова с одинаковым оригиналом.

Словарь сохраняется в формате xml.

Графический интерфейс пользователя

Графический интерфейс пользователя словаря состоит из следующих частей:

  • Виджет таблицы
  • Виджет ввода
  • Виджет поиска
  • Виджет добавления
  • Виджет редактирования
  • Виджет статистики

Виджет таблицы

Главное окно

Виджет таблицы отображает содержимое словаря.

Таблица содержит следующие столбцы:

Оригинал
Слово на исходном языке. Макс. длина: 50.
Транскрипция
Транскрипция слова. Макс. длина: 50.
Перевод
Перевод слова. Макс. длина: 100.
Статус
Степень освоения слова.
Дата
Дата добавления слова.

Виджет ввода

Главное окно

С Виджетом ввода вы можете быстро добавить новое слово. Достаточно указать оригинал слова и нажать на кнопку Добавить.

Вы также можете указать транскрипцию, перевод и статус для слова.

Виджет поиска

Меню → Изменить → Найти...

Виджет ввода → Найти

С помощью Виджета поиска вы можете задать фильтр для Виджета таблицы.

Введите фильтр в строке ввода и нажмите на кнопку Ввести. Для сброса фильтра очистите строку ввода и нажмите на кнопку Ввести.

Виджет добавления

Меню → Изменить → Добавить...

Виджет таблицы → ПКМ → Добавить...

Виджет добавления это окно, где вы можете указать для слова оригинал, транскрипцию, перевод, статус и комментарий и добавить его в словарь.

Виджет редактирования

Меню → Изменить → Изменить...

Виджет таблицы → ПКМ → Изменить...

С помощью Виджета редактирования вы можете редактировать группу слов.

Выберите необходимые строки в таблице и откройте Виджет редактирования. Если вы не укажете строки, будут выбраны все.

В появившемся окне вы можете редактировать или удалить слово. При необходимости вы можете восстановить исходное значение слова.

Для сохранения изменений нажмите кнопку Сохранить. Для внесения текущих изменений в таблицу нажмите на кнопку Записать. Для отмены всех изменений нажмите на кнопкуОтменить.

Виджет статистики

Меню → Инструменты → Статистика

Виджет статистики показывает статистику по вашему словарю.

Интерфейс командной строки

Используйте аргумент -h или --help для получения списка всех возможных опций:

podiceps -h
