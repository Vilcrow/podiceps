Shortcuts

Shortcut Action
Main Window
Ctrl + N create a new file
Ctrl + O open file
Ctrl + S save file
Ctrl + Shift + S save file as
Ctrl + I show information about the current file
Ctrl + W quit the program
Ctrl + Alt + Z undo the table action
Ctrl + Alt + Y redo the table action
Ctrl + Shift + A open the add window
Ctrl + F open the find widget
Ctrl + Shift + E open the words edit window
Delete delete selected rows
Ctrl + P open the preferences
Ctrl + Shift + N show the statistics
Esc deselect the table rows and clear the input
Input Widget
Enter add a word
Ctrl + Z undo (edit line)
Ctrl + Shift + Z redo (edit line)
Ctrl + X cut (edit line)
Ctrl + C copy (edit line)
Ctrl + V paste (edit line)
Find Widget
Enter set a filter
Esc clear the filter input line
Alt + F close the widget
Alt + O set the original column
Alt + R set the transcription column
Alt + T set the translation column
Alt + S set the status column
Alt + D set the date column
Alt + C set the comment column
Word Add
Alt + A add a word
Alt + C close the window
Word Edit
Alt + R record changes to the table and close the window
Esc don't record changes to the table and close the window
Enter save changes
Alt + D delete a word
Alt + C recover a word
Alt + P go to previous word
Alt + N go to next word
